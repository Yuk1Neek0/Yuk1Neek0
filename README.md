## Featured Project: Family Daily Bot

**Family Daily Bot** is my main AI application project: a Telegram-based family daily life logging assistant that uses AI to classify food, exercise, sleep, mood, and health-related messages, then generates daily and weekly reports.

It is built as a practical AI product, not just a chatbot demo.

### What it does

- Receives daily life logs through Telegram
- Classifies messages using AI
- Handles food, exercise, sleep, mood, and health-related records
- Generates daily, weekly, and monthly reports
- Produces report image cards using HTML rendering and Playwright
- Provides dashboard-style monitoring for logs, reports, routing, and AI behavior
- Uses model routing between different LLM providers
- Includes semantic routing and memory-style behavior for better classification and response quality

### Engineering highlights

- Python backend with Flask
- Telegram webhook integration
- PostgreSQL / Supabase database
- Dockerized deployment
- Gunicorn + Nginx + Cloudflare HTTPS
- APScheduler for recurring report jobs
- Playwright for report card rendering
- DeepSeek and Gemini model routing
- RAG-style memory and semantic routing concepts
- Structured development workflow with testing, documentation, and staged improvements

### Tech stack

`Python` · `Flask` · `PostgreSQL` · `Supabase` · `Docker` · `Nginx` · `Cloudflare` · `Telegram Bot API` · `APScheduler` · `Playwright` · `Gemini` · `DeepSeek` · `RAG` · `Semantic Routing`
